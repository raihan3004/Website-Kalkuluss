Penjelasan Fitur

  • Turunan : Digunakan untuk menghitung turunan pertama dari fungsi f(x). Pengguna juga dapat mengetahui nilai turunan pada suatu nilai x tertentu.
  • Integral : Digunakan untuk menghitung integral tak tentu dari fungsi f(x). Hasil integral ditampilkan dengan konstanta + C.
  • Limit : Digunakan untuk menentukan nilai limit fungsi saat x mendekati suatu nilai tertentu menggunakan pendekatan kiri dan kanan.

Web kalkulus ini dibuat sebagai alat bantu pembelajaran untuk memahami konsep dasar kalkulus seperti turunan, integral, dan limit, serta sebagai penerapan pemrograman web berbasis JavaScript.
